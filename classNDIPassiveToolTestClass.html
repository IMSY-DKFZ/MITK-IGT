Detailed Description

Documentation NDIPassiveTool has a protected constructor and a protected itkFactorylessNewMacro so that only it's friend class NDITrackingDevice is able to instantiate tool objects. Therefore, we derive from NDIPassiveTool and add a public itkFactorylessNewMacro, so that we can instantiate and test the class
