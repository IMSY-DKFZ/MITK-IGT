Detailed Description

ToFPMDRawPlayerDeviceFactory is an implementation of the factory pattern to generate Raw Player Devices. ToFPMDRawPlayerDeviceFactory inherits from AbstractToFDeviceFactory which is a MicroService interface. This offers users the oppertunity to generate new Raw Player Devices via a global instance of this factory.
